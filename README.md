🚀 Neo's Terminal Companion Bot

Official support bot for **Neo's Terminal**.

Neo's Terminal Companion Bot allows subscribers to communicate directly with the administrator, submit suggestions, report bugs, and provide feedback to help improve the Neo's Terminal community.

✨ Features

* 💡 Submit suggestions and ideas
* 🐞 Report bugs and issues
* 📩 Contact the administrator
* 📢 Send feedback about channel content
* 🤝 Improve communication between the community and the admin

📋 Commands

* `/start` — Start the bot
* `/help` — View available commands
* `/suggest` — Send a suggestion
* `/bug` — Report a bug
* `/contact` — Contact the administrator
* `/about` — Learn more about the bot
* `/channel` — Open Neo's Terminal

 🛠️ Technologies Used

* Java
* Maven
* Telegram Bot API
* IntelliJ IDEA
* Git & GitHub

 🎯 Purpose

The goal of this bot is to serve as the official support center for Neo's Terminal by providing an easy way for users to communicate with the administrator, report issues, and contribute ideas for future improvements.

 🔒 Security

Sensitive information such as bot tokens is stored using environment variables and is never committed to the repository.
